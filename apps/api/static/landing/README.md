Place approved public-landing hero artwork in this directory.

Preferred filenames:
- `hero-top.svg`
- `hero-bottom.svg`

Keep both files the same dimensions, ideally `1920x1080`.

Fallback filename:
- `hero.svg`

Alternative extensions are also supported for the preferred and fallback filenames:
- `.png`
- `.jpg`
- `.jpeg`
- `.webp`
- `.avif`
